hexo同步布署在github-coding并实现国内外自动分流

567899.xyz hexo

建议单独托管在github上，因为国内大环境背景下，你懂的啦！

域名解析

登陆DNSPOD控制台 在域名添加2条记录值

记录类型 线路类型 记录值
CNAME 国外 username.github.io
CNAME 国内 username.coding.me

配置文件

在_config.yml站点配置文件中添加配置：

# Deployment
# Docs: https://hexo.io/docs/deployment.html
deploy:
- type: git
  repo: git@github.com:username/username.github.io.git
  branch: master
- type: git
  repo: git@git.coding.net:username/username.coding.me.git
  branch: master

然后执行hexo deploy完成。
