下载安装oci命令行管理工具

bash -c "$(curl –L https://raw.githubusercontent.com/oracle/oci-cli/master/scripts/install/install.sh)"

一路回车就好了，结束使用下面命令查看安装是否成功

oci -v

oci setup config

接下来需要把公共秘钥传到控制台，回到浏览器，同样进入用户界面查看

oci iam availability-domain list

如果没有报错说明配置成功。
