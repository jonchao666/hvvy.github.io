    Github Telegram (https://doub.io) 账号分别都删库清空删号了 fork to (https://github.com/doubi)

    代理相关

    ss_go.sh

    • 脚本说明: Shadowsocks 一键安装管理脚本
    • 系统支持: CentOS6+ / Debian6+ / Ubuntu14+
    • 使用方法: https://doub.io/ss-jc67/
    • 项目地址: https://github.com/shadowsocks/go-shadowsocks2

    脚本特点: 目前网上的各个Shadowsocks脚本基本都是只有 安装/启动/重启 等基础功能，对于小白来说还是不够简单方便。既然是一键脚本，那么就要尽可能地简单，小白更容易接受使用！

    下载安装:

    wget -N --no-check-certificate https://raw.githubusercontent.com/ojbk8/doubi/master/ss-go.sh && chmod +x ss-go.sh && bash ss-go.sh
