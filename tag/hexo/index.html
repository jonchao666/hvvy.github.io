  • 简要安装hexo和next主题

    以下以Windows平台为例

    下载安装Node和git

    Git - Downloads 和 下载 | Node.js

    安装hexo

    打开Git Bash依次执行

    npm install hexo-cli -g
cd /d  #这里以D盘为例
hexo init blog
cd blog #/d/blog为hexo的根目录
npm install
hexo server

    应该就能在浏览器中打开(http://localhost:4000) 看到效果了。
